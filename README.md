# 🛍️ Customer Shopping Behavior Analysis

## 📖 Project Overview

This project is an end-to-end data analytics project that analyzes customer shopping behavior using transactional data from **3,900 purchases** across multiple product categories. The objective is to uncover customer spending patterns, product preferences, subscription trends, and purchasing behavior to support data-driven business decisions.

The project demonstrates the complete analytics workflow—from data cleaning and feature engineering in **Python**, to business analysis in **PostgreSQL**, and interactive dashboard creation in **Power BI**.

---

## 🎯 Objectives

- Clean and preprocess customer transaction data.
- Perform Exploratory Data Analysis (EDA).
- Create meaningful features for better analysis.
- Analyze business questions using SQL.
- Develop an interactive Power BI dashboard.
- Provide actionable business recommendations.

## 🧹 Data Cleaning & Preprocessing

The following preprocessing steps were completed using Python:

- Loaded data using Pandas.
- Explored the dataset using `info()` and `describe()`.
- Identified and handled missing values.
- Imputed missing **Review Rating** values using the median rating within each product category.
- Renamed columns to **snake_case**.
- Created new features:
  - `age_group`
  - `purchase_frequency_days`
- Removed redundant columns after validation.
- Loaded the cleaned dataset into PostgreSQL for SQL analysis.

---

## 📊 SQL Business Analysis

The following business questions were answered using PostgreSQL:

1. Revenue generated by gender.
2. High-spending customers who used discounts.
3. Top 5 highest-rated products.
4. Purchase comparison by shipping type.
5. Subscriber vs. non-subscriber spending analysis.
6. Products most dependent on discounts.
7. Customer segmentation (New, Returning, Loyal).
8. Top 3 products within each category.
9. Relationship between repeat purchases and subscriptions.
10. Revenue contribution by age group.

## 🚀 Key Insights

- Subscribers generated higher overall revenue than non-subscribers.

- Certain products relied heavily on discounts to drive purchases.
- Express shipping customers generally spent more per order.
- Revenue was concentrated within specific age groups, enabling targeted marketing opportunities.
